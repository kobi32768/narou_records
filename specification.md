# Narou records

小説家になろうでの読書量をカクヨムのように表示したい

## 用語

| 用語 | 定義 |
| -- | -- |
|ブックマーク|小説家になろうの純正ブックマーク|
|しおり(栞)|拡張機能で追加する独自で自動的に挟まれるもの|

## ほしい情報

* 累計文字数
* 冊数 ( 累計文字数 / 10万 )
* ( 月別 | 日別 ) 読了文字数
* ( 月別 | 日別 ) 作品数
* ( 月別 | 日別 ) エピソード数

## 動作

### 累計文字数

重複してカウントすることがないようにしなければならない
ブックマークリストの全作品から累計文字数を取得
差分で計算しないと計算量が多そう

1～ブックマーク位置 の各エピソード文字数の総和
最新話まで読んだ作品は計算せず文字数取得

ログイン必須情報だけど大丈夫かな……？
